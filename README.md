# SOP Manager Application
SOP Manager is an XWiki application designed to manage the lifecycle of official procedures and standard operating documents within an organization.
It allows users to create, categorize, review, and approve procedures directly in XWiki, ensuring that documents follow a controlled process before publication.
Once approved, procedures can be exported as PDF files and published to a designated location, such as XWiki File Manager or the document page itself, where they can be distributed, downloaded, and signed for official use.
This helps organizations maintain clear governance, traceability, and consistent access to the latest approved procedures.
